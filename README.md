# Vue Responsive Search
### A Responsive Client side search with filters made using Vue.js, Bootstrap4, Flexbox. 
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
